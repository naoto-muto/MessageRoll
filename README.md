# MessageRoll

### 機能

- テキストファイルを読み込んで画面に文字をスクロール表示。
- テキストファイルが追記されたら、追記されたタイミングで表示。

### 動作確認環境

- Windows 10 
- .NETFramework 4.6.1
- Visual Studio 2017

### 使い方

`あああ` を表示した場合(デフォルト設定)

```bat
rem MessageRoll.exe のカレントディレクトリで実行
echo あああ >> watch.txt
```
